# Automatic plant watering system

This project was made for me to learn about embedded programming on
an ESP32 board. The idea is to connect it to a submergible capacitive
sensor and a water pump to read inputs and actuate the pump whenever
the plant needs water.

Additionally, I have extra temperature and humidity sensors, that
I might just connect to the board and send data through WiFi to the
instance of Home Assistant I have running at home. I'm not sure if
this will be out of scope though.
